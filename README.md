### `Tech`

- Python 3.11
- Wagtail 5.2
- Django 4.2

### `Setup`

#### `Linux`

Install pyenv following the insturctions [here](https://gist.github.com/trongnghia203/9cc8157acb1a9faad2de95c3175aa875)

#### Install python 3.11

```shell 
pyenv isntall 3.11
```

#### Install pipx

```shell
sudo apt install pipx
pipx ensurepath
```

#### Install poetry

```shell
pipx install poetry
```

#### Install python packages

```shell
poetry install --no-root
```
